### JSFL 打包脚本
***
### 如何使用
> 1. 前提安装好flash pro，双击脚本可直接运行。<br/>
> 2. 选中一个文件夹，将整个文件夹下的全部图片按目录结构导入新建的fla文件，自动添加类定义，导出swf，等价于一个打包压缩文件的脚本。

Adobe官方JSFL帮助文档： http://help.adobe.com/zh_CN/flash/cs/extend/index.html
